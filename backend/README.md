# Setup Environment
## Windows
```
python3 -m venv venv
.\venv\Scripts\activate.ps1
pip install -r requirements.txt
```

## Linux
```
python3 -m venv venv
source /venv/bin/activate
pip install -r requirements.txt
```

## MacOS
```
python3 -m venv venv
source /venv/bin/activate
pip install -r requirements.txt
```

# Run the Server
```
python main.py
```
